# Smart Fit - Challenge

Repo with the guide [front-end challenge](https://github.com/bioritmo/front-end-code-challenge-smartsite)

## Production

Open [https://smartfit-mancf.netlify.app/](https://smartfit-mancf.netlify.app/) with your browser to see live.
## Development
First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.

## Tests
run the tests

```bash
npm run test
# or
yarn test
```
